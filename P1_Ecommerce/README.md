# 📦 Análisis de Ventas E-commerce

Análisis exploratorio de datos (EDA) sobre un dataset de transacciones
de una tienda online con más de 500,000 registros, usando Python y Pandas.

![Evolución de ventas](notebooks/ventas_mensuales.png)

---

## 🎯 Objetivo

Identificar patrones de ventas, productos más rentables y comportamiento
de clientes para apoyar decisiones comerciales basadas en datos.

## ▶️ Cómo ejecutar

1. Clona el repositorio:
```bash
   git clone https://github.com/tuusuario/eda-ecommerce
```
2. Instala las dependencias:
```bash
   pip install pandas matplotlib seaborn jupyter
```
3. Abre el notebook:
```bash
   jupyter notebook notebooks/analisis.ipynb
```
